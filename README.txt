** Two apps formed and synced with Microsoft Dynamics **
* Drawing features to sketch in classroom.
* Taking classroom notes section which is synced to One Note.
* Classroom Boards section which is the notice board of the classroom.
* Homework section where the homework is synced to Microsoft Planner.
* Attendance record of the student.
* Live engaging games to evaluate the students.
* Report of the students using visualisation tools such as pie chart, bar graph e.t.c
